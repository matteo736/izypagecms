## Informazioni su IzyPage

Un CMS moderno e leggero sviluppato con Laravel, Inertia.js e React, pensato per competere con WordPress. **Attualmente in fase MVP** (Minimum Viable Product) con queste caratteristiche principali:
- 🚀 Stack tecnologico moderno
- ⚡ Prestazioni ottimizzate
- 🔐 Sicurezza avanzata
- 🧩 Architettura modulare
- 👥 Gestione ruoli utente

**Funzionalità MVP:**
- ✔️ Sistema di gestione pagine
- ✔️ Sistema articoli/blog
- ✔️ Gestione utenti e ruoli
- ✔️ URL e metadati SEO-friendly
- ✔️ Design responsive
- ✔️ Media library di base

## Funzionalità Principali

### 🖌 Gestione Pagine (MVP)
- Creazione/modifica pagine con editor WYSIWYG
- Sistema di routing dinamico
- Campi per ottimizzazione SEO
- Versionamento delle pagine

### 📝 Gestione Contenuti (MVP)
- Creazione/modifica articoli
- Sistema categorie e tag
- Pubblicazione programmata
- Bozze e revisioni

### 👤 Gestione Utenti (MVP)
- Controllo accessi basato su ruoli (RBAC)
- Autenticazione/registrazione utenti
- Reset password
- Log attività

### 🛠 Pannello Admin (MVP)
- Dashboard con statistiche
- UI reattiva con React
- Aggiornamenti in tempo reale
- Monitoraggio sistema

## Stack Tecnologico
- **Backend**: Laravel 10+
- **Frontend**: React 18 + Inertia.js
- **Database**: MySQL/PostgreSQL
- **Auth**: Laravel Breeze
- **UI Kit**: Shadcn/UI
